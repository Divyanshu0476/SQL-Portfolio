# Window Functions in SQL – LEAD() Practice

This file contains SQL queries demonstrating the use of window functions,
especially LEAD(), for analytical problem solving.

## Concepts Covered
- Row-wise comparison
- Salary difference analysis
- Ranking using DENSE_RANK()
- Percentage change calculation

## Use Cases
- Salary benchmarking
- Sales trend analysis
- Growth rate calculation

## Key Insight
Window functions allow comparing rows without collapsing the dataset,
making them essential for analytical SQL.
